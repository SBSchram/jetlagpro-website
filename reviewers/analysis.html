Reproduce This Analysis

All analyses shown on this page can be independently reproduced. Download the raw data and analysis script to verify our statistical methods and results.

Step 1: Download Raw Trip Data

Export all trip completion records (including survey responses) from our public Firestore database:

curl -s "https://firestore.googleapis.com/v1/projects/jetlagpro-research/databases/(default)/documents/tripCompletions?pageSize=1000" \ -o trips.json
Data Included: Each trip record contains:
  • Trip details: ID, dates, airports, time zones, direction
  • Intervention data: Points stimulated, completion method
  • Survey responses: Pre-travel and post-travel symptom ratings (fatigue, sleep disruption, cognitive function, mood, digestive issues)
  • Metadata: HMAC signatures, timestamps, device info

Note: If the response includes "nextPageToken", repeat the call with &pageToken=... to retrieve all pages.

Step 2: Download Analysis Script

Get the standalone Python analysis script (pure Python, minimal dependencies):

curl -o analyze_jetlag_data.py \ https://jetlagpro.com/scripts/analyze_jetlag_data.py

View Source Code on GitHub

Step 3: Run the Analysis

Execute the analysis script to reproduce all statistics shown on this page:

python analyze_jetlag_data.py \ --trips trips.json \ --output analysis_report.txt

Step 4: Compare Results

The script will generate a detailed report (analysis_report.txt) containing:

  • Basic Statistics ‚Äì Trip counts, average points stimulated, average severity
  • Dose-Response Analysis ‚Äì Mean severity by time zone and stimulation level
  • Point Usage Statistics ‚Äì Frequency of each acupuncture point

Compare the output with the live data displayed on this page. Values should match (within rounding).
