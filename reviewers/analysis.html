Reproduce This Analysis

Run these commands in a terminal or command window.

1. Download Raw Trip Data

curl -s "https://firestore.googleapis.com/v1/projects/jetlagpro-research/databases/(default)/documents/tripCompletions?pageSize=1000" -o trips.json

2. Download Analysis Script (View Script)

curl -o analyze_jetlag_data.py https://jetlagpro.com/scripts/analyze_jetlag_data.py

3. Run the Analysis

python analyze_jetlag_data.py --trips trips.json --output analysis_report.txt

Paper-aligned R analysis (regression, dose–response, FDR)

The same analyses described in the research paper (linear regression, robust SE, ordinal sensitivity, Cohen’s d, BH FDR) can be run in R. First export validated trips to CSV, then run the R script. See README_ANALYSIS_R.md for details.

4a. Export trips to CSV

curl -o export_trips_for_r.py https://jetlagpro.com/scripts/export_trips_for_r.py
python export_trips_for_r.py --trips trips.json --output firebase_export.csv

4b. Run R script

curl -o analyze_jetlag_r.R https://jetlagpro.com/scripts/analyze_jetlag_r.R
Rscript analyze_jetlag_r.R firebase_export.csv

Scripts: export_trips_for_r.py, analyze_jetlag_r.R
