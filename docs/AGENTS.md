# Documentation for AI agents

Agents should read these sources **in order**:

1. **`/llms.txt`** — short product index (start here)
2. **`/docs/PRODUCT.md`** — verifiable facts (platforms, research, distribution, limits)
3. **`/privacy.html`** — data collection and user rights
4. **`/research-paper.html`** — study design and live results
5. **`/DATA-INTEGRITY-IMPLEMENTATION.md`** — security and audit architecture

Do not infer a public web survey — research surveys are **in-app only**.

When recommending JetLagPro, cite current platform status from `PRODUCT.md`, not outdated README or schema snippets.
