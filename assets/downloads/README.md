# Research paper exports (PDF / Word-friendly HTML)

The files here are generated by the build script. From the **website repo root** run:

```bash
npm install
npm run export-paper
```

**Output:**
- `JetLagPro_Research_Paper.pdf` — open in any PDF viewer.
- `JetLagPro_Research_Paper_for_Word.html` — open in Microsoft Word, then **File → Save As → Word Document (.docx)** to get an editable Word file. We do not generate .docx directly; Word opens this HTML and saves it as .docx with full fidelity.

**First time:** run `npx playwright install` to install the browser. Re-run after changing `research-paper.html` so the downloads stay in sync.
