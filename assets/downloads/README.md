# Research paper exports (PDF)

The files here are generated by the build script. From the **website repo root** run:

```bash
npm install
npm run export-paper
```

**Output:**
- `JetLagPro_Research_Paper.pdf` — open in any PDF viewer.

**First time:** run `npx playwright install` to install the browser. Re-run after changing `research-paper.html` to refresh the PDF export.
